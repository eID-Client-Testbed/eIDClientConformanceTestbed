Copy certificates from generated TestObject (tests\[TO_Example]\certificategeneration\certs\) to CartSimControl (CartSimControl\certs).

[Step 2]:

[Step 5]:
Run cardSim_*.bat corresponding to TR-03124 testcase.

Start simulator testcase.
