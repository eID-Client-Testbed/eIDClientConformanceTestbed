Voraussetzungen:

1. Java 8 SDK installieren.
2. Cygwin installieren
3. Maven installieren (siehe https://maven.apache.org/download.cgi#Installation).
3.1 Maven bin Verzeichnis in PATH Variable aufnehmen.
3.2 JAVA_HOME Variable setzen (auf das installierte JDK Hauptverzeichnis).
3.3 Maven konfigurieren

Build erstellen:

0. (optional) build_from_source.sh mit dos2unix in Cygwin-Terminal konvertieren.
1. build_from_source.sh in Cygwin-Terminal ausführen
2. SpecificTestObject manuell erzeugen und beifuegen.
